# もじトレ モンスター ✏️🐉

ひらがな・カタカナ・学年別漢字（小1〜小6）の **書き順** を、タッチペンや指で
なぞって練習できる、子ども向けの Web アプリです。雑になぞって線をはみ出すと
「アウト」になるので、正しい書き順・運筆が自然と身につきます。

## あそびかた
1. 「だれが やる？」で名前を登録（きょうだいなど、何人でも別々に記録）
2. 「れんしゅう する」→ ひらがな／カタカナ／学年を選ぶ → 文字を選ぶ
3. お手本の線を **正しい書き順・向き** でなぞる（はみ出すとアウト！）
4. 5回れんしゅう → ヒントなしの **テスト** に合格でクリア＝けいけんちゲット
5. けいけんちで **相棒モンスター** が育つ・進化する（レアなほど育成は大変）

## 特長
- **完全自己完結の1ファイル**（外部通信なし・オフライン可・データ収集なし）
- 進捗は端末内（localStorage）にのみ保存。サーバーに送信しません
- iPad / Chromebook / PC ブラウザ対応（タッチ・ペン・マウス）
